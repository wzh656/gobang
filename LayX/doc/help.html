1. 如何在iframe页面中关闭当前窗口？ 

parent.layx.destroy(id);   // 方法一（推荐）
top.layx.destroy(id);   // 方法二
用 还是 取决于 layx.js的引用位置。

2. 如何在iframe页面中动态获取窗口 id？ 

function getLayxId() {
     var id;
     if (self != top && self.frameElement && self.frameElement.tagName == "IFRAME") {
         var layxWindow = $(self.frameElement).parents(".layx-window");
         id = layxWindow.attr("id").substr(5);
     }
     return id;
 }

3. 如何动态设置iframe窗口大小？ 

parent.layx.setSize(getLayxId(),{width:1000,height:500});        // 在iframe页面调用
